About Me

I used to be an aspiring writer (maybe I still am?) but now I’m trying my hand at web design. And although I’m still very much a novice when it comes to coding, I’m learning new things all the time and absorbing as much as I can. I hope to take the skills I learn and parlay them into an exciting career path as a front-end web designer, making beautiful and easily navigable pages for people to enjoy.

Almost my entire time in higher academia was spent honing my craft as a creative writer. And while I still have an immense appreciation for prose and poetry, I’m slowly becoming fluent in the languages of HTML and CSS as well.

Goals

I want to master the process of building web sites and increase my knowledge, skills and abilities in:

  • HTML
  • CSS
  • JavaScript
  • React

I would like to work for a web design firm and help create websites that clients are proud of.
